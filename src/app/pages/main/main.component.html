Welcome to eFlight

Summary

eFlight is a repository where you'll find some very useful apps that us, pilots, use in our day-to-day operations.

For the main funcionality of this app, you must import the Lido PDF flight plan.

Very simple to access and use, they can help you to tackle some of the tasks we are dealt everyday, like calculating long haul rest periods, checking fuel figures, etc.
